La prise de note ( PENDANT )

Il existe pleins de moyen de rendre plus rapide la prise de note :

  • Employer des abréviations,des symboles,...
  • Employer des listes bien espacés
  • Employer des couleurs

Il faut faire la prise de note sur le verso de la feuille !

La prise de note ( APRÈS )

Il y a plein de chose à faire après une prise de note SUR LE RECTO DE LA FEUILLE :

  • Noter les définitions
  • Faire un plan du cours ( la structure, les différents titres )
  • Résumé les différents idées principales
